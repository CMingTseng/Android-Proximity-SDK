ProximityZoneRangeBuilder

interface ProximityZoneRangeBuilder

A fluent builder API sub-class for configuring ProximityZone's trigger distance.

Functions

inCustomRange

 abstract fun inCustomRange(desiredMeanTriggerDistance: Double): ProximityZoneBuilder

Zone's onEnter/onExit actions will be invoked in the custom range. The precision is difficult to predict due to a nature of the radio waves, so the actual invocation might occur within the defined custom range with a few meters margin.

inFarRange

 abstract fun inFarRange(): ProximityZoneBuilder

Zone's onEnter/onExit actions will be invoked in the FAR range. The precision is difficult to predict due to a nature of the radio waves. Basically, the FAR range should vary between 4 - 7 meters from a beacon.

inNearRange

 abstract fun inNearRange(): ProximityZoneBuilder

Zone's onEnter/onExit actions will be invoked in the NEAR range. The precision is difficult to predict due to a nature of the radio waves. Basically, the NEAR range should vary between 0 - 2 meters from a beacon.
