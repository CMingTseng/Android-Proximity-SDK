inCustomRange

abstract fun inCustomRange(desiredMeanTriggerDistance: Double): ProximityZoneBuilder

Zone's onEnter/onExit actions will be invoked in the custom range. The precision is difficult to predict due to a nature of the radio waves, so the actual invocation might occur within the defined custom range with a few meters margin.
